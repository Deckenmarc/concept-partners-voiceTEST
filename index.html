Voice Agent Demo

Test eis zwee Voice Agents mat deem selwechte Szenario fir ze kucken wéi een am beschten reagéiert.

Agent 1

Dësen Agent ass den éischten Test. Klickt op de Widget fir mat him ze schwätzen an ze testen wéi hien op Är Froen reagéiert.

Tipp: Probéiert deeselwechte Szenario mat béide Agents aus fir ze vergläichen.

ELSA 2 TEST

Dësen Agent ass den zweeten Test. Benotzt deeselwechte Szenario wéi beim éischten Agent fir ze kucken wéi een besser funktionéiert.

Tipp: Notéiert Ären Androck vun all Agent fir se ze vergläichen.

Note: Set RETELL_PUBLIC_KEY and RETELL_AGENT_ID environment variables in your deployment.
